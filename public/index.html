HOW TO PLAY

PLAYER 1

Use QWER keys to answer:

  • Q - Green option
  • W - Blue option
  • E - Red option
  • R - Yellow option

PLAYER 2

Use UIOP keys to answer:

  • U - Green option
  • I - Blue option
  • O - Red option
  • P - Yellow option

RULES

  • First player to answer gets the points.
  • Answer quickly to get more points.
